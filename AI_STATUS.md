# AI Chatbot - Важная Информация ⚠️

## Текущая Ситуация

AI чатбот работает, но **Gemini API имеет квоту запросов**. Сейчас вы видите ошибку потому что:

1. **Квота исчерпана** - Бесплатный API ключ имеет лимит запросов в минуту
2. **Решение временное** - Добавлены fallback ответы

## Что Сделано

✅ **Улучшена обработка ошибок**:
- Чатбот теперь показывает полезное сообщение вместо "ошибка"
- Добавлены контакты поддержки в fallback ответе
- Генератор описаний создает базовый шаблон при ошибке

## Для Production Использования

### Вариант 1: Увеличить Квоту (Рекомендуется)
```
1. Перейти на платный план Gemini API
2. Или использовать другой API ключ с большей квотой
```

### Вариант 2: Добавить Rate Limiting
Ограничить количество запросов от одного пользователя:
```javascript
// В server/middleware/rateLimiter.js
const aiLimiter = rateLimit({
    windowMs: 60 * 1000, // 1 минута
    max: 3, // 3 запроса в минуту
    message: "Слишком много запросов к AI. Попробуйте через минуту."
});
```

### Вариант 3: Кэширование Ответов
Сохранять популярные вопросы/ответы в базе данных.
