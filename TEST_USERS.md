## 🚀 Как Использовать

### Создание Пользователей

```bash
cd server
node seed-users.js
```

### Вход в Систему

1. Откройте: http://localhost:5173/login
2. Введите email из списка выше
3. Пароль: `password123`
4. Нажмите "Войти"

### Доступ к Панелям

**Администраторы:**
- Перенаправление: `/admin`
- Доступ ко всем функциям

**Партнеры:**
- Перенаправление: `/admin` (Partner Dashboard)
- Управление товарами, заказами, клиентами

**Покупатели:**
- Перенаправление: `/profile`
- Покупки, избранное, заказы
